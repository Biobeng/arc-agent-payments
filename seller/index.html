How it works

Payment Flow

x402 + EIP-3009
01 Agent sends a request to a paid endpoint Buyer Agent
02 Seller returns HTTP 402 with payment requirements Seller API
03 Agent signs an EIP-3009 authorization offchain Buyer Agent
04 Agent retries with payment signature attached Buyer Agent
05 Gateway verifies and settles the payment Circle Gateway
06 Seller serves the response. No gas paid per call. Seller API

Try it

Run the buyer agent

TypeScript

Clone the repo, add your buyer wallet private key to .env, and run the agent. It will fund itself from Gateway, make paid API calls autonomously, and print a cost summary.

# Clone and install git clone https://github.com/Biobeng/arc-agent-payments cd arc-agent-payments/buyer npm install # Add your buyer private key echo "PRIVATE_KEY=0xYOUR_KEY" > .env # Run the agent npm run run-agent
